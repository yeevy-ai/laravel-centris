# Security Policy

## Supported versions

The latest 0.x release receives security fixes.

## Reporting a vulnerability

Report privately via [GitHub Security Advisories](https://github.com/yeevy-ai/laravel-centris/security/advisories/new) — do **not** open a public issue.

You'll get an acknowledgment within a few days; please allow a reasonable window for a fix before any disclosure.

**Never include real Passerelle feed data or FTP credentials in a report** — reproduce with synthetic data.
